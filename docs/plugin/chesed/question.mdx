# Q&A

解释一些问题。

## 这是什么插件？

Chesed 是一款功能全面的附魔分离插件，您可以通过卸魔台，花费经验等级、游戏币或点券来拆卸装备上的附魔。

## Chesed 与 EnchantSeparator 有什么区别？

Chesed 是 EnchantSeparator 的升级版，它的性能更好，功能更多，也更易于使用。

## EnchantSeparator 可以直接升级到 Chesed 吗？

除自定义菜单配置外，其他数据均支持从 EnchantSeparator 一键迁移到 Chesed，详见 [数据迁移](start/migrate)。

## 支持什么服务端？

目前我们支持 1.14 到 1.21.1 的全部版本，支持 Spigot、Paper 和 Folia 服务端。

## 我可以使用其他更多附魔插件吗？

可以，目前我们支持 Aiyatsbus、EcoEnchants 和 NereusOpus。

## 卸魔支持花费金币吗？

可以，我们目前提供了 Vault、PlayerPoints 和原版经验等级。如您需要更多货币类型，我们也提供了 API 来支持，请联系作者。
