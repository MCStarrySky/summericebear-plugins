# Q&A

解释一些问题。

## Aiyatsbus 与 NereusOpus 有什么区别？

Aiyatsbus 是 NereusOpus 的升级版，其前身是没有写完的 SplendidEnchants。Aiyatsbus 是脱离于 NereusOpus 代码完全重写的版本，它的性能更好，也更易于使用。

## NereusOpus 可以直接升级到 Aiyatsbus 吗？

可以，并且 Aiyatsbus 提供了迁移功能，可以自动迁移 NereusOpus 的附魔配置、变量数据。

## 我可以从其他附魔插件迁移吗？

不完全支持，有一些注意事项，详见 [数据迁移](start/migrate)。

## 我不想花钱，我能用吗？

可以，但是你需要自己编译插件，且没有任何服务支持。
