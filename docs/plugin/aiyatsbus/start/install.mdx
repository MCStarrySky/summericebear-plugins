# 安装插件
将插件本体放入服务端插件目录下，并重启服务端。服务器启动过程中你将看到如下相关信息。

:::tip

Aiyatsbus Docs 中的演示环境为 git-Purpur-2138 (MC: 1.20.4)

:::

```
[Aiyatsbus] Generated ModernEnchantmentRegisterer in 0ms
[Aiyatsbus] Generated AiyatsbusMinecraftAPI in 392ms
[Aiyatsbus] Loading server plugin Aiyatsbus v0.7.0
[Aiyatsbus] Enabling Aiyatsbus v0.7.0
[Aiyatsbus] 良好 | 8 个附魔品质已加载 (0 ms)
[Aiyatsbus] 良好 | 21 个附魔对象已加载 (126 ms)
[Aiyatsbus] 良好 | 110 个附魔已加载 (768 ms)
[Aiyatsbus] 良好 | 10 个附魔组已加载 (2 ms)
```
若启动过程中没有产生任何关于 Aiyatsbus 的错误信息，则表示插件已安装成功。
