# 属性元数据

这部分内容涉及附魔的属性数据，也就是品质和对象。

```yaml
rarity: 普通
targets:
  - 弓
  - 弩
  - 三叉戟
```

## 节点介绍

| 节点            | 描述                       |
|-----------------|----------------------------|
| `rarity`        | 附魔品质，不写则为默认值 |
| `targets`        | 附魔对象，不写则无法应用至任何物品 |

## 附魔品质

目前，Aiyatsbus 默认提供如下附魔品质：

- 普通
- 罕见
- 精良
- 史诗
- 传奇
- 稀世
- 诅咒
- 皮肤

默认品质为 `普通`，当你将品质错写成一个不存在的品质，或没有配置附魔品质时都将配置为默认品质。

:::tip

你可以在主要配置文件 `config.yml` 中更改默认品质。

:::

## 附魔对象

指定了附魔在哪些物品上生效，你可以填写这些物品的中文名，也可以填写英文名，但注意部分名称需写为复数形式。

:::tip

有些物品的名称和官方译名不一致，例如 `锹` 和 `铲`。你可以去附魔对象配置中更改中英文名称。

:::

目前我们提供的附魔对象如下。

| 中文名 | 英文名 |
|-------|-------|
| 剑 | `swords` |
| 斧 | `axes` |
| 重锤 | `maces` |
| 镐 | `pickaxes` |
| 铲 | `shovels` |
| 锄 | `hoes` |
| 弓 | `bow` |
| 弩 | `crossbow` |
| 三叉戟 | `trident` |
| 头盔 | `helmets` |
| 胸甲 | `chestplates` |
| 护腿 | `leggings` |
| 靴子 | `boots` |
| 鞘翅 | `elytra` |
| 剪刀 | `shears` |
| 盾牌 | `shield` |
| 钓鱼竿 | `rod` |
| 打火石 | `flint_and_steel` |
| 萝卜钓竿 | `carrot_stick` |
| 头饰 | `headable` |
| 可损坏物品 | `damageable` |
| 全部物品 | `all` |
