# 变量配置

这部分内容涉及附魔的变量配置，附魔的变量总共分为四种类型

## 节点介绍

| 节点  |  英文名           | 描述                       |
|-----------------|----------------------------|
| 等级相关 | `leveled`        | 与等级强相关，可通过附魔等级计算数值                   |
| 玩家相关 | `player_related`| 与玩家强相关，可使用 PlaceholderAPI 变量计算数值           |
| 物品相关 | `modifiable`            | 与物品强相关，可读写物品 NBT/PDC 数据实现成长型附魔 |
| 常量 | `ordinary`          | 不变的参数，一般情况可视作附魔配置或功能开关配置                    |

详见后续页面。
