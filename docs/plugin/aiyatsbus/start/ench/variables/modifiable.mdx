# 物品变量

这部分内容涉及附魔的与物品相关的变量配置，你可以读取物品 `NBT` 或 `PDC` 中的数据。

:::tip

例如你想读取/写入武器的击杀数，就可以配置此类变量。

:::

配置方式如下。

```yaml
variables:
  modifiable:
    当前累计: test_current_total=0
    是否充能完毕: (NBT)can_discharge=充能中
```

变量分为三部分，以 `当前累计: test_current_total=0` 举例，可拆成 `当前累计`、`test_current_total`、`0` 三部分。

|  部分         |  示例           | 描述                       |
|------------------------|-----------|-----------------|
| 变量名称    | `当前累计`        | 该变量名称，同时也作为该配置项的名称，必须在该附魔的全部变量名中保持唯一                   |
| 节点名称    | `test_current_total`| 节点名称，若开头为 `(NBT)` 则读取物品 NBT 相应节点，否则读取 PDC          |
| 初始数值    | `0`            | 该变量初始数值，若物品上没有对应节点则返回此数值，用等号分割变量名称和初始数值 |

:::tip

NBT 节点支持深度读取，使用 `.` 作为分层符来指定嵌套结构中的键。

:::
