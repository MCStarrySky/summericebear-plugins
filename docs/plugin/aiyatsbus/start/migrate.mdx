# 数据迁移

你可以从其他插件迁移附魔数据到 Aiyatsbus，但这种操作会有一定的风险，且有一些数据无法迁移。

## NereusOpus

使用指令 `/aiyatsbus api migrate nereusopus` 进行迁移。

该操作会读取 NereusOpus 数据文件夹内的所有附魔，并且读取每个附魔的基本数据，例如是否开启、禁用世界等。并且，如果您没有开启 NereusOpus 的自动平衡性调整，Aiyatsbus 会尝试迁移所有的附魔变量等数据。

## EcoEnchants

由于 EcoEnchants 策略，最新版本的插件暂无法迁移，但 EcoEnchants 8 可以。

:::tip

事实上 NereusOpus 大部分附魔就是从 EcoEnchants 8 而来。我们没有做单独的 EcoEnchants 8 附魔包。

:::

## AdvancedEnchantments

我们制作了适配 AdvancedEnchantments 的附魔包，您可以直接安装，但不支持导入变量数据。

:::tip

我们还没有做完全部的附魔。

:::
