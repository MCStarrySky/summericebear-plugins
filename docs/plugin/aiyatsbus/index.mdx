# 前言

欢迎来到 Aiyatsbus Docs。

这里定义了基于 [Aiyatsbus](https://github.com/Micalhl/Aiyatsbus) `1.0` 实现自定义附魔而提供的指导性准则和建议。

Aiyatsbus 是一个为 Minecraft（Java 版）提供更多附魔的平台，它的前身是 NereusOpus。但是 Aiyatsbus 不仅仅只是提供了更多的附魔，还提供了一个令人惊叹的框架来创建属于您自己的独一无二的附魔，例如提供了一些可以操作事件、方块、实体或其他东西的魔法般的工具，通过我们自己的脚本语法帮助您向游戏中添加更多的乐趣。

- 在 2.x 版本中，NereusOpus 的自定义附魔过于繁琐且复杂，不过现在您可以通过 Aiyatsbus 轻松创造属于自己的附魔。
- 我们使用 GPL-3.0 协议进行开源, 这意味着您可以通过编程来创建属于自己的特性。
- 免费 (提供 **有限的** 服务)。

随着 Aiyatsbus 的更新，我们认为 2.0 的代码过于繁琐且不智能，这已经变得很麻烦了。我们需要的是一个用高效的框架精心重写、高度自定义、稳定、易于使用的插件。因此迎来大规模更新已成定局。我们重新精心设计了 NereusOpus 的每一个组件，重新设计了一套框架，Aiyatsbus 应运而生。

通过此部分教程，您可以基本了解 Aiyatsbus 的运作方式以及配置方式，掌握初步的自定义附魔技能。
