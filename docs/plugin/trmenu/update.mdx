# 版本更新

目前，Dreeam-qwq 分支的 TrMenu 尚未支持 1.21.1。

该分支 (StarrySky Fork) 已更新至最新的 TabooLib，并提供了最高到 1.21.1 版本的支持。

您可以通过以下教程来迁移您的 TrMenu。

## 1.20.4 及以下版本

不管您是用的是什么分支，该分支都尚未支持 1.21.1。

您可以直接安装我们开发组发布的 3.3.1-beta1 及以上版本的 TrMenu，可以获得完整的功能支持。

## 1.20.5 及以上版本

若您的服务器为 1.20.5 及以上版本，您应该正在使用的是 lilingfeng 提供的 TrMenu。

您需要对现有的服务器环境进行一些调整才可以使 TrMenu 更好的支持您所运行的服务器。

- 重新打开发包模式（关闭 TrMenu 配置中的 Static-Inventory）。
- 若您没有其他插件使用 NBT-API，则可删除。
- 可以使用 NBT，不再有限制。
- 头颅正常。
- 若插件稳定，则一般没有报错，如有报错则一定是不正常的，需要反馈至 [Issues](https://github.com/MCStarrySky/TrMenu/issues)。

::: caution

由于 1.20.5 对 NBT 的改动，您目前只能获取到各个插件自定义的 NBT 以及 PDC。

:::
