# EnchantIntensify - 附魔强化

## ♟ 插件定位
本插件提供了有关附魔的优化机制，为玩家提供了诸如强化卷轴（直接强化某附魔）、幸运石（提升强化成功率）、辟邪石（降低强化失败反噬率）等道具，便于玩家进一步优化和提升附魔，也为各位服务器管理员开辟了新市场。

## ♛ 插件特色
- 自由设定强化上限与具体概率
- 自由选择强化机制，如装备破碎，反噬，DNF机制（等级越高，难度越高）等
- 自由设定概率，支持公式化设定
- 高自由度权限设定，强化成功公告设定

## ☞ 实装效果

### 强化卷轴

:::info

强化卷轴分中级、高级两种。玩家获得的中级卷轴成功率为0-80%，高级卷轴为0-100%。

:::info
