# 指令大全
## 帮助命令
:::info

/et help<br/>
查询附魔强化系统的帮助。

:::
## 强化道具仓库查看
:::info

/et gui `<player>`<br/>
为某玩家打开强化道具仓库。具有op权限的玩家可直接拿取各类物品。

:::
## 强化台
:::info

/et intensify `<player>`<br/>
为某玩家打开附魔强化台。

:::
## 强化道具发放
:::info

/et give `<ID>` `<amount>` `<player>`<br/>
将一定数量强化道具给予某玩家。<br/>
其中`<ID>`即为[saved_items.yml](saved_items.yml)中所配置的每个物品的内部存储名。

:::
