# 附魔分离-插件简介
> 为何选择我们？

## ♟ 插件定位
EnchantSeparator插件为服务器附魔系统提供了全新的拆卸附魔机制——通过使用有关道具，消耗特定货币，即可在卸魔系统中进一步优化各类装备的附魔。

## ♛ 插件特色
### 稳定的卸魔机制
玩家在使用附魔台的时候，装备上出现一些弱的附魔是很难避免的。一些服务器中，玩家一件装备上的附魔数量有限，在这种情况下，有什么办法能使有限的附魔带来的效益最大化呢？

附魔分离插件的主要功能——卸魔台，能精准地卸下装备众多附魔中的某一个，从而移除低质量附魔，解决了原版的难题。

不仅如此，玩家还可以将装备上的优质附魔卸下，成为附魔书的形式，去强化别的装备。
